## Install :

```
$> bundle
```

## Preview :

```
$> jekyll serve
```

## Deploy :

```
$> octopress deploy
```

## New post :
```
$> octopress new post "My Title"
```

## Change ruby version:

### List of Ruby versions:
```
$> rvm list
```

### Install new version
```
$> rvm install ruby-2.3
```

And edit the `.ruby-version` with the new version.
